AntiMicroX is an open-source project licensed under GPL-3.0 and welcomes contributions of all kinds — bug fixes, new features, translations, and documentation improvements.

## Before you start

<Steps>
  <Step title="Check existing issues">
    Browse the [issue tracker](https://github.com/AntiMicroX/antimicrox/issues) to see if the bug or feature you have in mind is already reported or discussed.
  </Step>
  <Step title="Open a discussion for new features">
    If you want to propose a new feature, [open a new issue](https://github.com/AntiMicroX/antimicrox/issues/new/choose) first to discuss the idea before writing code.
  </Step>
  <Step title="Read the coding guidelines">
    Review the [coding guidelines on the wiki](https://github.com/AntiMicroX/antimicrox/wiki/Coding-guidelines) before writing code. Document your code in Doxygen-compatible format where applicable. The [online Doxygen docs](https://antimicrox.github.io/doxygen/) are useful for understanding the codebase structure.
  </Step>
  <Step title="Submit a pull request">
    Fork the repository, make your changes on a branch, and submit a pull request. For larger changes, open a **Draft Pull Request** first to show your direction and get early feedback.
  </Step>
</Steps>

## Code contributions

All contributions are subject to the project's GPL-3.0 license. By submitting a pull request, you agree to license your contribution under those terms.

<Tip>
Check the issue tracker for issues with bounties — these are marked to attract contributors and may include compensation for the work.
</Tip>

## Translations

Translations are managed through [Weblate](https://hosted.weblate.org/engage/antimicrox/). To contribute a translation:

1. Visit the [AntiMicroX project on Weblate](https://hosted.weblate.org/engage/antimicrox/)
2. Select your language (or request a new one)
3. Translate strings directly in the browser — no local setup required

More information about the translation process is available on the [Translating AntiMicroX wiki page](https://github.com/AntiMicroX/antimicrox/wiki/Translating-AntiMicroX).

## Sharing profiles

If you have created a useful gamepad profile for a specific game or application, you can share it in the [antimicrox-profiles repository](https://github.com/AntiMicroX/antimicrox-profiles). Community profiles can also be found there if you are looking for a starting point for a new game.

## Getting help

- Use the [GitHub Discussions](https://github.com/AntiMicroX/antimicrox/discussions) tab for questions about the codebase.
- The [online Doxygen API docs](https://antimicrox.github.io/doxygen/) document internal classes and their relationships.
- Upvote issues with 👍 to signal which bugs or features matter most to you — this helps maintainers prioritize.
